# Flutter x Firebase

**Specification :**
- Flutter 1.12.13
- Latest Firebase Auth Plugin
- Latest Google Sign In Plugin
- Latest Firestore Plugin

**What it capable of :**
- Login Using Google Sign In
- Showing User Account in Drawer
- Sign Out
- Showing Data from Firebase in Card View
- Able Editing Data
- Able Delete Data
